O que é o CSS?

Sintaxe: (clique aqui para abrir a imagem grande)

Exemplo de sintaxe: (clique aqui para ver mais de um exemplo)

A sintaxe é constituída pelo seletor, que é a tag do HTML a ser modificada, a tag escolhida vai ter as modificações que vai ser programada no CSS, o seletor tem as chaves, que delimita o inicio e final do código, a propriedade a ser mudada deve ser explicitada entre as chaves e ter os dos dois pontos no final “:” para definir o valor, após ser definido o valor deve ser fechada com o ponto e vírgula “;” os comentários são definidos entre os caracteres “/* */”; Quando se tem uma subpropriedade tem que colocar-lha logo após a propriedade separada por um traço “-”.


Formas do CSS

O CSS tem três formas de ser utilizado, sendo elas “Inline” que é definido dentro da linha da tag, o “Incorporado” que é definido dentro da tag head, o “CSS” definido nele só vale para a página atual, e por ultimo o “Externo” que é um documento a parte, você escreve o “CSS” somente uma única vez e pode executar em qualquer página apenas fazendo um link com o documento externo. As prioridades que o navegador dá para cada estilo é a ordem respectivamente feita com a explicação, o Inline, Incorporado e Externo respectivamente.
