Como se Organiza

    Para se organizar um documento é fácil, primeiro faça uma pasta mãe, a que vai conter todas as outras pastas, após a criação da pasta principal, faça uma pasta para o CSS e outra para o JS caso necessário, faça também a pasta para as imagens ou qualquer outra coisa que você queira colocar no seu site, os documentos HTML ficam na pasta mãe, eles NÃO ficam em nenhuma subpasta.
